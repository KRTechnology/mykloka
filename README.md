# Kloka - Employee Management System

Kloka is an employee management system built for Kimberly Ryan, featuring attendance tracking and task management capabilities.

## Features

- User Authentication & Authorization
- Role-based Access Control
- Attendance Tracking (Clock In/Out)
- Task Management
- Employee Dashboard
- Administrative Controls

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Drizzle ORM
- Vercel Postgres
- NextAuth.js
- Nodemailer
- React Email
- Shadcn/ui
- Framer Motion

## Getting Started

1. Clone the repository
2. Copy `.env.local.example` to `.env.local` and fill in the environment variables
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
